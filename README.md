# pad-img

Utility that converts portrait images to a wallpaper with a specific padding color

### Built With

*   python
*   Docker
*   bash

## Getting Started
* Local install
  ```bash
     git clone https://github.com/gsiri007/bg-wall.git
     cd bg-wall
     ./setup.sh
  ```
* Docker install
  ```bash
     git clone https://github.com/gsiri007/bg-wall.git
     cd bg-wall
     ./docker-setup.sh
  ```

### Prerequisites

* Local setup
   * git
   * python
   * pip
* Docker setup
   * git
   * Docker

## Build upon 

* Python Imaging Library: [Pillow (PIL Fork)](https://pypi.org/project/pillow/)
